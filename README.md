# Solace Engine

## A 3D Game Engine

## Quick Start

### Requirements
- CMake 3.12+
- C++ 20 compatible compiler


### Clone the repository:
```bash
git clone https://github.com/JamieSchooling/Solace.git
cd Solace
```

### Build and Run:
<details>
<summary>Windows</summary>

**Build:**
```bash
cmake -B build
cmake --build build --config Release
```

**Run:**
```bash
build\Demo\Release\Demo.exe
```
</details>
